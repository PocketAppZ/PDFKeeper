Document Record Keywords

The Keywords (read only) text box on the main form is where the Keywords from the document record are displayed. These keywords are read from the PDF document when uploaded to the database.

Before continuing, if the Search Results view is displayed across the entire main form; select the View menu, and then select Toggle Right Panel; or select the Toggle Right Panel tool bar icon; or press F12 on the keyboard.

Viewing Keywords:

  1. Select the document record in the Search Results view.

  2. Select the Keywords tab.
