Uninstalling PDFKeeper

PDFKeeper can be uninstalled from the computer by using Programs and Features in Windows. The uninstall can be performed as a standard Windows user, without elevated privileges.

The following are NOT removed by the uninstall process:

  • %LOCALAPPDATA%\Robert_F._Frasca\PDFKeeper.exe_StrongName_3a4kmi4qmybqtr3ccktl5hjos3hxcjc5

  • %APPDATA%\Robert F. Frasca\PDFKeeper

  • HKEY_CURRENT_USER\Software\Robert F. Frasca\PDFKeeper\AutoUpdater

  • HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Robert F. Frasca\PDFKeeper

  • Database installation and client software.
