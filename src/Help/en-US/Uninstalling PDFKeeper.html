Uninstalling PDFKeeper

PDFKeeper can be uninstalled from the computer by using Programs and Features in Windows. The uninstall can be performed as a standard Windows user, without elevated privileges.

The following folders, including any files and folders contained within will not be removed:

%LOCALAPPDATA%\Robert_F._Frasca

%APPDATA%\Robert F. Frasca

In addition, the database installation and any database client software will remain untouched.

If the querying of all document records was disabled by policy, then the following registry key will remain and need to be removed with elevated privileges:

HKEY_LOCAL_MACHINE\SOFTWARE\Robert F. Frasca
