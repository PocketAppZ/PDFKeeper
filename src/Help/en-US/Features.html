Features

  • PDF documents are stored and indexed in a compatible, relational database where they can be managed by any computer that has PDFKeeper installed on your network.

  • PDF documents can be uploaded individually or in bulk, and can be uploaded using configured folders to allow for integration and automation.

  • The Flag Document state can be set on a selected document record and when PDF documents are uploaded as a way to mark for follow-up.

  • Notes can be added to a selected document record that can include the date, time, and database user account name. All notes can be edited and are indexed by the database.

  • Document records can be searched by String, Author, Subject, Author and Subject, or Date Added. In addition to search capabilities, the entire database can be queried or only query document records that are flagged.

  • For a selected document record, the following functions are available: PDF document viewing with the bundled or default viewer; Flag Document state management; Notes viewing and editing; Keywords, PDF Preview image, and PDF Text are also displayed for viewing.

  • PDF documents with their notes and flag state can be exported from the database for easy importing.
