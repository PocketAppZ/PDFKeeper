Setup Client-Server Database

PDFKeeper can be setup to use a compatible, client-server relational database for document storage. It is recommended that the relational database management system be installed on a dedicated computer system that is capable of handling the expected load.

At this time, PDFKeeper has only been designed for client-server use with Oracle Database. Future releases may include support for additional relational database management systems based on user request and the feasibility of implementation. For a database to be considered, please open a new issue on the Issue Tracker.

As an alternative to the fully supported, paid editions of Oracle Database, Oracle offers an easy to install, Oracle Database Express Edition (XE) which is perfect for small work group environments that do not need a fully supported edition or for those who just wish to try out PDFKeeper – all completely free!
