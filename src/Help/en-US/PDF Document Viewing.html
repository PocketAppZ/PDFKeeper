PDF Document Viewing

To view the PDF document associated with a document record, perform any of the following methods:

  • Double click on the document record in the Search Results view.

  • Select the document record in the Search Results view, and then select the File menu, and then select Open.

  • Select the document record in the Search Results view, and then press Alt + F, and then O; or press Ctrl + O on the keyboard.

  • Select the document record in the Search Results view, and then select the Open tool bar icon.

Additional Information:

  • By default, the PDF document is opened with the bundled, third-party viewer. To open with the user's default PDF application, see this topic.
