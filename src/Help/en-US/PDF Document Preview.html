PDF Document Preview

The Preview (read only) picture box on the main form is where the first page from the selected PDF document is displayed as an image.

Before continuing, if the Search Results view is displayed across the entire main form; select the View menu, and then select Toggle Right Panel; or select the Toggle Right Panel tool bar icon; or press F12 on the keyboard.

Previewing PDF Document:

  1. Select the document record in the Search Results view.

  2. Select the Preview tab.

Setting the Preview Image Resolution:

The resolution of the image (in dots per square inch) that is displayed in the picture box can be changed. This will have a zoom in and out effect on the image being displayed.

To change the image resolution DPI, perform the following:

  1. Double click on the Preview picture box; or select the View menu, and then select Set Preview Image Resolution; or press Ctrl + I on the keyboard.

  2. On the Preview Image Resolution dialog, set the Preview Image Resolution DPI to a value from 10 to 600 using the arrows on the control with the mouse or by using the up and down arrows on the keyboard.

  3. Select OK or press Enter on the keyboard. After doing so, the image will be redrawn at the new DPI.
