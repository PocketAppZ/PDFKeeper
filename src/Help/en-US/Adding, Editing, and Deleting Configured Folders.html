Adding, Editing, and Deleting Configured Folders

The Upload Folders form is where configured Upload folders are added, edited, and deleted.

To add a new folder:

  • Select Add or press Alt + A on the keyboard. After doing so, the [Upload Folder Configuration form will open].

To edit an existing folder:

  • In the list, select the folder that you wish to edit, and then select Edit or press Alt + E on the keyboard. After doing so, the [Upload Folder Configuration form will open].

To delete an existing folder:

  • In the list, select the folder that you wish to delete, and then select Delete or press Alt + D on the keyboard.
