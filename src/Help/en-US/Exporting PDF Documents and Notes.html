Exporting PDF Documents and Notes

To export the PDF document and corresponding notes from one or more document records returned from a search, perform the following steps:

  1. On the Search Results view, click on the check box for each record that you wish to delete. To select all documents in the Search Results view, select the File menu, then Select, and then All; or press Alt + F, then C, and then A on the keyboard.

  2. Select the File menu, and then select Export; or press Alt + F, and then E on the keyboard.

  3. Select the folder location where to create the export folder that will contain the exported PDF documents and corresponding notes text files.

  4. Select OK or press Enter on the keyboard to perform the export.

Additional Information:

  • You can navigate the Search Results view using the up and down arrows on the keyboard and check/uncheck a document record using the space bar on the keyboard.

  • To uncheck all document records in the Search Results view, select the File menu, then Select, and then None; or press Alt + F, then C, and then N on the keyboard.

  • The absolute path name of the folder where the exported files are located will be displayed in a message box after the export has completed.

  • Exporting is a convenient way to transfer PDF documents and notes to another database system without having to use the source and target database systems export/import processes. The export will create a folder that contains each exported PDF document, named PDFKeeper<ID>.pdf and corresponding text file, named PDFKeeper<ID>.txt that contains the notes stored in the database for that document.

  • When exporting documents for transfer to another system, it is highly recommended that nobody else be logged into PDFKeeper.

  • To import the folder containing the exported PDF documents and corresponding notes, login to PDFKeeper using the database system that the PDF documents and text files will need to be imported into, and then copy the entire folder using Windows Explorer into PDFKeeper Upload, located in the Windows Documents folder. The PDF and text files will be imported during the next upload cycle.
