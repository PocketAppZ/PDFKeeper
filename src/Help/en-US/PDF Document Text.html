PDF Document Text

The Text (read only) text box on the main form is where the plain text, extracted from the PDF document prior to uploading is displayed.

Before continuing, if the Search Results view is displayed across the entire main form; select the View menu, and then select Toggle Right Panel; or select the Toggle Right Panel tool bar icon; or press F12 on the keyboard.

Viewing Text:

  1. Select the document record in the Search Results view.

  2. Select the Text tab.

Saving Text to a text file:

  1. Select the document record in the Search Results view.

  2. Select the Text tab, and then select the text box inside the Text tab page.

  3. Select the File menu, and then select Save As; or press Alt + F, and then E on the keyboard.

  4. Select the folder location and enter the filename to save as.

  5. Select Save or press Enter on the keyboard.

Printing Text:

  1. Select the document record in the Search Results view.

  2. Select the Text tab, and then select the text box inside the Text tab page.

  3. Select the File menu, and then select Print; or select the Print tool bar icon; or press Ctrl + P on the keyboard. As an option, to preview before printing, select the File menu, and then select Print Preview; or press Alt + F, and then V on the keyboard.

Additional Information:

  • Text contained in images is not supported and will not be displayed.

  • No formatting is applied to the extracted text.
