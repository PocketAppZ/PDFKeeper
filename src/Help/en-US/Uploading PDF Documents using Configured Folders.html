Uploading PDF Documents using Configured Folders

PDF documents can be uploaded using configured folders from outside of PDFKeeper using the following methods:

  • From Web Browser/application, save the PDF document into the desired sub-folder inside PDFKeeper Upload, located in the Windows Documents folder.

  • Using Windows Explorer, copy the PDF document(s) that you wish to upload and paste into the desired sub-folder inside PDFKeeper Upload, located in the Windows Documents folder.

Additional Information:

  • PDFKeeper must be open for the PDFKeeper Upload folder shortcut to be available in the Windows Documents folder.

  • The selected sub-folder located in PDFKeeper Upload must have been created by PDFKeeper.

  • The Title, Author, Subject, Keywords, and flag document state will be applied to the PDF document(s) using the configuration for the selected sub-folder.

  • USER password protected PDF documents are not supported by PDFKeeper. If you have the legal right to remove the password, you can find plenty of information on the Internet to assist you.

  • To automate the uploading of PDF documents, see Automating with the PDFKeeper Upload Process.
