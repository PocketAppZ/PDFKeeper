Installing ODP.NET Managed Driver

This topic provides the steps that need to be performed to install ODP.NET, Managed Driver that is required for PDFKeeper to connect to Oracle Database.

If ODP.NET, Managed Driver is already installed, this topic can be skipped; however, it is recommended to use the latest version. If upgrading to the latest version, make sure to uninstall the existing version as instructed by the ODP.NET, Managed Driver Installation Instructions.

  1. Download the latest xcopy version of the ODP.NET, Managed Driver, then unblock and unzip the file.

  2. For installation instructions, refer to readme.htm that is included with the xcopy ZIP file. Note, when installing on a UAC (User Access Control) enabled Windows system, make sure to perform the installation with a Command Prompt session that was opened using Run as administrator. In addition, when installing, the machine_wide_configuration argument must be true to install into the Global Assembly Cache (GAC).
