Configuring an Upload Folder

The Upload Folder Configuration form is where the Information Properties are defined for the upload folder being added or edited.

  1. Enter or edit the Folder Name.

  2. Enter, edit, or select the Title Prefill.

  3. Enter, edit, or select the Author Prefill. The Author Prefill combo box is linked to the database to allow for selecting an existing Author.

  4. Enter, edit, or select the Subject Prefill. The Subject Prefill combo box is linked to the database to allow for selecting an existing Author.

  5. Enter or edit the Keywords Prefill (optional).

  6. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to search for bills that will require payment.

  7. To save the configuration, select OK or press Enter on the keyboard. To discard the changes, select Cancel or press Esc on the keyboard.
