Installing Oracle Data Provider for .NET

This topic provides the installation steps for Oracle Data Provider for .NET that is required for PDFKeeper to connect to Oracle Database.

This version of PDFKeeper requires ODP.NET, Managed Driver 4.122.21.1 that is included with Oracle Data Access Components 21.4 to be installed. Older versions of ODP.NET, Managed Driver and Oracle Data Access Components used by previous versions of PDFKeeper should be uninstalled if no longer needed by other applications.

  1. Go to the ODAC Downloads page.

  2. Under ODAC Xcopy Packages, select Earlier versions of ODAC 21c.

  3. Under Software select Oracle Data Access Components 21.X.

  4. Under Platforms select Microsoft Windows x64 (64-bit).

  5. Click the zip file link next to Oracle Data Access Components 21.4 Xcopy for (Microsoft Windows x64 (64-bit)) to download.

  6. Open the folder where the zip file was downloaded to, and then open the file properties of the zip file and select Unblock.

  7. Unzip the file into a staging folder.

  8. For installation instructions, refer to readme.htm that is included in the zip file.

Additional Information:

  • Make sure to perform the installation using a Command Prompt session that was opened using Run as administrator.

  • It is not necessary to install the full ODAC product, only ODP.NET4 is required.
