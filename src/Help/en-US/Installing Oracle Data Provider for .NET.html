Installing Oracle Data Provider for .NET

This topic provides the installation steps for Oracle Data Provider for .NET that is required for PDFKeeper to connect to Oracle Database.

This version of PDFKeeper requires ODP.NET, Managed Driver 4.122.19.1 that is included with Oracle Data Access Components (ODAC) Runtime 19.3 to be installed. Starting with PDFKeeper 6.0.0, the 64-bit Oracle Data Access Components (ODAC) Runtime 19.3 is required. Older versions of ODP.NET, Managed Driver and 32-bit Oracle Data Access Components (ODAC) Runtime 19.3 used by previous versions of PDFKeeper, can be uninstalled if no longer needed by other applications.

  1. Go to ODAC Runtime Downloads.

  2. Download 64-bit ODAC 19.3 from the ODAC Xcopy section to a folder for staging.

  3. In the properties of the zip file downloaded, select Unblock, and then unzip the file.

  4. For installation instructions, refer to readme.htm that is included in the zip file.

Additional Information:

  • Make sure to perform the installation using a Command Prompt session that was opened using Run as administrator.

  • It is not necessary to install the full ODAC product, only ODP.NET4 is required.
