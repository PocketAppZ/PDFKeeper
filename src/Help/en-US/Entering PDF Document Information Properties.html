Entering PDF Document Information Properties

The Add PDF Document form is where the Information Properties for the selected PDF document are edited, and then applied to a modified version of the selected PDF document that is staged for uploading.

  1. To view the selected PDF document with the bundled viewer, select View Original or press Alt + V on the keyboard.

  2. Enter the PDF document Title. To set the Title to the file name of the selected PDF document, select Set to File Name or press Alt + F on the keyboard.

  3. Enter or select the Author of the PDF document. The Author combo box is linked to the database to allow for selecting an existing Author.

  4. Enter or select the Subject of the PDF document. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  5. If necessary, enter document Keywords.

  6. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to search for bills that will require payment.

  1. Check or uncheck Delete original PDF document when OK is selected. When checked, the original PDF document is deleted to the Windows Recycle Bin.

  2. Select Save or press Alt + S on the keyboard to write the modified version of the original PDF document that includes the specified PDF Information Properties to %APPDATA%\Robert F. Frasca\PDFKeeper\UploadStaging. If the original PDF document contained an OWNER password, the password will not be written to the modified PDF document. Selecting Save will need to be repeated if changes are made following a Save and prior to selecting OK.

  3. If necessary, you can view the modified version of the selected PDF document with the bundled viewer either by selecting Preview or by pressing Alt + P on the keyboard.

  4. If no further changes need to be made and you wish to upload the file, select OK or press Enter on the keyboard. To cancel and discard the modified PDF document, select Cancel or press Esc on the keyboard.

Additional Information:

  • Upload processing will be paused while the Add PDF Document form is open.
