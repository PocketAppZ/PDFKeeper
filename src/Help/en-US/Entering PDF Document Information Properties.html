Entering PDF Document Information Properties

The Add PDF Document form is where the Information Properties for the selected PDF document are edited, and then applied to a clone of the selected PDF document that is staged for uploading.

  1. To view the selected PDF document with the bundled viewer, either select View Original or press Alt + V on the keyboard.

  2. Enter the PDF document Title. To set the Title to the file name of the selected PDF document, select Set to File Name or press Alt + F on the keyboard.

  3. Enter or select the Author of the PDF document. The Author combo box is linked to the database to allow for selecting an existing Author.

  4. Enter or select the Subject of the PDF document. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  5. If necessary, enter document Keywords.

  6. Refer to Document Information Properties Editing and Uploading in original help file to continue this topic.
