Uploading a PDF Document from within PDFKeeper

  1. From the PDFKeeper main form, select the File menu, and then select New; or select the New tool bar icon; or press Ctrl + N on the keyboard.

  2. Browse and select the PDF document you wish to upload, and then select Open or press Enter to display the Add PDF Document form where PDF Document Information Properties are entered.

Additional Information:

  • If the selected PDF document contains an OWNER password, the correct password will need to be entered when prompted. If you do not know the password and have the legal right to remove the password, you can find plenty of information on the Internet to assist you.
