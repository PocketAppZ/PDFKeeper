PDFKeeper Options

The following options can be changed from the Options dialog:

Select last row when displaying Search Results.

  • When checked, the last row in the Search Results view will be selected following a search or a refresh.

  • When unchecked, the first row in the Search Results view will be selected following a search or a refresh.

Open PDF documents with default application.

  • When checked, any PDF document associated with a document record in the Search Results view will be opened using the bundled, third-party viewer.

  • When unchecked, any PDF document associated with a document record in the Search Results view will be opened using the user's default PDF application.

Do not upload PDF 1.7 documents.

  • When checked, any PDF document with a version of 1.7 will not be uploaded to the database. This option is provided as Oracle Text in Oracle Databases prior to version 12c do not index PDF 1.7 documents resulting in the document not being searchable by text contained within the document.
