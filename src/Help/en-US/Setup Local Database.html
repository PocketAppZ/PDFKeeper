Setup Local Database

SQLite is the relational database engine used by PDFKeeper for local document storage. It provides an easy to setup, single user database option that does not require a relational database management system.

The local database is contained in a single file named PDFKeeper.sqlite that is created in %APPDATA%\Robert F. Frasca\PDFKeeper. This database will need to be backed up regularly in the event the database needs to be restored due to corruption or accidental data loss. You are fully responsible for this database!
