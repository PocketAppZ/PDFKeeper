Searching by Text

Document records can be searched for by text, meaning any plain text that was extracted from each PDF document when uploaded to the database, including metadata, text annotations, and any notes added to the document record after uploading. When the PDF document being uploaded is “image-only” and the operating system is Windows 10, the text will be extracted from the PDF document using OCR.
