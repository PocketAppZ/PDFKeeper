Adding, Editing, and Deleting Upload Folder Configurations

The Manage Upload Folder Configurations form is where Upload Folder Configurations are added, edited, and deleted.

To add a new folder configuration:

  • Select New or press Alt + N on the keyboard. After doing so, the Upload Folder Configuration section will enable.

To edit an existing folder configuration:

  • From the drop down list, select the folder configuration that you wish to edit, and then select Edit or press Alt + E on the keyboard. After doing so, the Upload Folder Configuration section will enable.

To delete an existing folder configuration:

  • From the drop down list, select the folder configuration that you wish to delete, and then select Delete or press Alt + D on the keyboard.

Additional Information:

  • Upload Folder Configuration XML files are stored in %APPDATA%\Robert F. Frasca\PDFKeeper\UploadConfig.

  • Upload folders are stored in %APPDATA%\Robert F. Frasca\PDFKeeper\Upload.
