Database Setup

Oracle Database

This topic provides the steps that need to be performed for first time setup of Oracle Database or Oracle Database Express Edition (XE) for use with PDFKeeper. It is assumed that Oracle Database has been installed on the computer that will host the database, you have familiarized yourself with the documentation specific to the Oracle Database edition and version being used, and assume full responsibility of administrating an Oracle Database.

  1. If Oracle Database was installed on a different computer, you will need to download Oracle Instant Client Basic or Basic Lite and SQL*Plus packages to this computer, extract the ZIP files into an empty folder, and then add that folder to the PATH. As an alternative, you can install PDFKeeper on the same computer as the Oracle Database and perform the setup on that computer.

  2. Run the Database Schema Setup.

Additional Information:

  • If the message “This page can’t be displayed” appears in the Help window after selecting Database Schema Setup in step 2, then restart this computer and try again.
