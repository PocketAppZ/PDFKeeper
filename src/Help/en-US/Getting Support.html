Getting Support

Support for PDFKeeper is provided through the Issue Tracker where all problems and feature requests are to be submitted. Before creating an issue, please make sure the problem or feature request has not already been submitted; and if it has, please update the existing issue.

When opening an issue for a problem, please make sure to provide, the exact steps to reproduce it, the version of PDFKeeper installed, and operating system version.

Note:

PDFKeeper logs all unhandled exceptions to:

<SystemDrive>\Users\<User>\AppData\Roaming\Robert F. Frasca\PDFKeeper\<version>\PDFKeeper.log

When opening an issue for an unhandled exception, please attach the log file.
