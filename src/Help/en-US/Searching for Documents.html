Searching for Documents

From the main form, PDF document records can be searched by String, Author, Subject, Author and Subject, or Date Added. In addition to search capabilities, the entire database can be queried or only query document records that are flagged.

How do you want to search for documents?

Search by String:

  1. Select the Search by String tab.

  2. Enter the search string or select a previously entered search string. For examples, see Searching by String Examples.

  3. Select Search or press Alt + S or Enter on the keyboard. After doing so, any document records found will be displayed in the Search Results view.

Search by Author:

  1. Select the Search by Author tab.

  2. From the drop down, select the Author. After doing so, any document records found will be displayed in the Search Results view.

Search by Subject:

  1. Select the Search by Subject tab.

  2. From the drop down, select the Subject. After doing so, any document records found will be displayed in the Search Results view.

Search by Author and Subject:

  1. Select the Search by Author and Subject tab.

  2. From the upper drop down, select the Author.

  3. From the lower drop down, select the Subject. After doing so, any document records found will be displayed in the Search Results view.

Search by Date Added:

  1. Select the Search by Date Added tab.

  2. From the drop down, select the Date. After doing so, any document records found will be displayed in the Search Results view.

Query All/Flagged Documents:

  1. Select the All/Flagged Documents tab.

  2. To query only flagged documents, check Flagged documents only.

  3. Select Query Documents or press Alt + Q on the keyboard. After doing so, all document records or flagged document records will be displayed in the Search Results view.

Additional Information:

  • In a multi user environment, it may be necessary to restrict who can query all documents. In some environments, querying an entire database table can have an adverse effect on database performance. For instructions, please see, Disable Querying of all Document Records by Policy.
