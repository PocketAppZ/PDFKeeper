Searching for Documents

From the main form, PDF document records can be searched by Text, Selections (Author/Subject/Category), or Date Added. In addition, the user can list document records that are flagged or all document records in the database.

How do you want to search for documents?

By Text:

  1. Select the Documents by Text Search function.

  2. Enter the search string or select a previously entered search string. For examples, see Search by Text Examples.

  3. Select Search by Text or press Alt + B on the keyboard. After doing so, any document records found will be displayed in the Search Results view.

By Selections (Author/Subject/Category):

  1. Select the Documents by Selections Search function.

  2. Start by selecting either an Author, Subject, or Category.

  3. If you wish to refine your search, select one of the remaining drop down boxes which will only contain items that match the item selected in the first drop down.

  4. If you wish to refine your search further, select the remaining drop down box which will only contain items that match the item selected in the first drop down and second drop down.

  5. Select Search by Selections or press Alt + Y on the keyboard. After doing so, any document records found will be displayed in the Search Results view. To clear search selections and start over, select Clear Selections or press Alt + C on the keyboard.

By Date Added:

  1. Select the Documents by Date Added Search function.

  2. From the drop down, select the Date. After doing so, any document records found will be displayed in the Search Results view.

Flagged Documents:

  • Select the Flagged Documents Search function. After doing so, any document records found will be displayed in the Search Results view.

All Documents:

  • Select the All Documents Search function. After doing so, any document records found will be displayed in the Search Results view.

Additional Information:

  • When the database contains document records that are flagged, PDFKeeper will display a red flag on the right hand side of the status bar.

  • In a multi user environment, it may be necessary to restrict who can search all documents. In some environments, searching an entire database table can have an adverse effect on database performance. For instructions, please see, Disable All Documents Search Function by Policy.
