Selecting and Adding PDF Documents to PDFKeeper

The Add PDF Documents form is where PDF documents can be selected and the Information Properties for the selected document are edited, and then applied to a modified version of the PDF that is staged for uploading.

  1. Select a PDF document to be uploaded by selecting Select or pressing Alt + E on the keyboard.

  2. Browse and select the PDF document you wish to upload, and then select Open or press Enter.

  3. To view the input PDF document with the bundled viewer, select View or press Alt + V on the keyboard.

  4. Enter the PDF document Title. To set the Title to the file name of the selected PDF document, select Set Title to File Name or press Alt + F on the keyboard.

  5. Enter or select the Author of the PDF document. The Author combo box is linked to the database to allow for selecting an existing Author.

  6. Enter or select the Subject of the PDF document. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  7. If necessary, enter document Keywords.

  8. If necessary, enter or select the Category of the PDF document. The Category combo box is linked to the database to allow for selecting an existing Category.

  9. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to search for bills that will require payment.

  10. Check or uncheck Delete input PDF document when OK is selected. When checked, the input PDF document is deleted to the Windows Recycle Bin.

  11. Select Save or press Alt + S on the keyboard to write the modified version of the PDF document that includes the specified PDF Information Properties to %APPDATA%\Robert F. Frasca\PDFKeeper\UploadStaging. If the input PDF document contained an OWNER password, the password will not be written to the modified PDF document. Selecting Save will need to be repeated if changes are made following a Save and prior to selecting Add.

  12. If necessary, you can view the modified version of the selected PDF document with the bundled viewer either by selecting Preview or by pressing Alt + P on the keyboard.

  13. If no further changes need to be made and you wish to upload the file, select Add or press Alt + A on the keyboard. To cancel and discard the modified PDF document, select Discard or press Alt + C on the keyboard.

Additional Information:

  • If the selected PDF document contains an OWNER password, the correct password will need to be entered when prompted. If you do not know the password and have the legal right to remove the password, you can find plenty of information on the Internet to assist you.

  • Upload processing will be paused after a PDF document has been selected and will be resumed after Add or Discard has been selected.
