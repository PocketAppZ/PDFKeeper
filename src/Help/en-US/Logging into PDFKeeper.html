Logging into PDFKeeper

On the PDFKeeper Login form:

  1. Enter your database User name.

  2. Enter your database Password.

  3. Enter the Data source specified as host:port.

  4. Select OK or press Enter.

Notes

  • host is the host name or IP address of the computer hosting the database. If connecting to the database located on the same computer, the hostname or IP address would be localhost or 127.0.0.1.

  • port (optional) is the TCP port number on which the Oracle Net listener is listening. If not specified, the default port number 1521 is assumed.
