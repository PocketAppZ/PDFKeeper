Document Record Notes

The Notes text box on the main form is where information related to the selected document record is entered. In addition, each entry can be stamped with the Date and Time, and the database user account name. All Document Notes are indexed when saved, making the text searchable.

Before continuing, if the Search Results view is displayed across the entire main form; select the View menu, and then select Toggle Right Panel; or select the Toggle Right Panel tool bar icon; or press F12 on the keyboard.

Editing Notes:

  1. Select the document record in the Search Results view.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. To add a stamp with the Date and Time, and the database user account name; either select the Edit menu, and then select Date/Time; or select the Date/Time tool bar icon; or press Ctrl + T on the keyboard.

  4. After making the necessary edits, save the changes by either selecting the File menu, and then select Save; or select the Save tool bar icon; or press Ctrl + S on the keyboard.

Saving Notes to a text file:

  1. Select the document record in the Search Results view.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. Select the File menu, and then select Save As; or press Alt + F, and then A on the keyboard.

  4. Select the folder location and enter the filename to save as.

  5. Select Save or press Enter on the keyboard.

Printing Notes:

  1. Select the document record in the Search Results view.

  2. Select the Notes tab, and then select the text box inside the Notes tab page.

  3. Select the File menu, and then select Print; or select the Print tool bar icon; or press Ctrl + P on the keyboard. As an option, to preview before printing, select the File menu, and then select Print Preview; or press Alt + F, and then V on the keyboard.

Additional Information:

  • When editing Notes, the Search Results view is disabled.

  • If you need to restore the text box contents back to the state when the document record was selected, basically undoing all of the edits made, select the Edit menu, and then select Restore; or select the Restore tool bar icon; or press Ctrl + R on the keyboard.
