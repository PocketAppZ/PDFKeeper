Sumatra PDF

https://www.sumatrapdfreader.org/

Sumatra PDF is included with PDFKeeper in unmodified, binary form for displaying PDF documents.

It is used in restricted mode for previewing a PDF prior to adding to the database; and used in unrestricted mode for viewing PDF documents returned from a search except when the user chooses to use their default application.
