Change Flag Document State

The Flag Document menu item on the main form is used to change the flag document state for the selected document record. When the Flag Document menu item is checked, the selected document record will be flagged.

Before continuing, if the Search Results view is displayed across the entire main form; select the View menu, and then select Toggle Right Panel; or select the Toggle Right Panel tool bar icon; or press F12 on the keyboard.

Change Flag Document state:

  1. Select the document record in the Search Results view.

  2. Select the Edit menu, and then select Flag Document; or press Alt + E, and then F on the keyboard.
