Set Category on Document Records

To set the category on one or more document records returned from a search, perform the following steps:

  1. On the Search Results view, click on the check box for each record that you wish to set the category on. To select all documents in the Search Results view, select the File menu, then Select, and then All; or press Alt + F, then C, and then A on the keyboard.

  2. Select the File menu, and then select Set Category; or press Ctrl + Shift + C on the keyboard.

  3. On the Set Category dialog, select the desired category or leave blank to clear, and then select OK or press Enter on the keyboard to perform the update.

Additional Information:

  • You can navigate the Search Results view using the up and down arrows on the keyboard and check/uncheck a document record using the space bar on the keyboard.

  • To uncheck all document records in the Search Results view, select the File menu, then Select, and then None; or press Alt + F, then C, and then N on the keyboard.
