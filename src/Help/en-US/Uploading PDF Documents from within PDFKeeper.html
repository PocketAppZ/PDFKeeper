Uploading PDF Documents from within PDFKeeper

For each PDF that you wish to upload, perform the following steps:

  1. From the PDFKeeper main form, select the File menu, and then select Add; or select the Add tool bar icon; or press Ctrl + A on the keyboard.

  2. Browse and select the PDF you wish to upload, and then select Open or press Enter.

  3. If the selected PDF contains an OWNER password, the correct password will need to be entered when prompted. If you do not know the password and have the legal right to remove the password, you can find plenty of information on the Internet to assist you.

  4. The Add PDF form will open.

  5. To view the selected PDF with the bundled viewer, select View or press Alt + V on the keyboard.

  6. Enter the PDF Title. To set the Title to the file name of the selected PDF, select Set Title to File Name or press Alt + F on the keyboard.

  7. Enter or select the Author of the PDF. The Author combo box is linked to the database to allow for selecting an existing Author.

  8. Enter or select the Subject of the PDF. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  9. If necessary, enter PDF Keywords.

  10. If necessary, enter or select the Category of the PDF. The Category combo box is linked to the database to allow for selecting an existing Category.

  11. If necessary, select the Tax Year of the PDF.

  12. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to search for bills that will require payment.

  13. Check or uncheck Delete selected PDF when added. When checked, the selected PDF is deleted to the Windows Recycle Bin.

  14. Select Save or press Alt + S on the keyboard to write the modified version of the PDF that includes the specified Information Properties and extension properties XML to %APPDATA%\Robert F. Frasca\PDFKeeper\UploadStaging. If the selected PDF contained an OWNER password, the password will not be written to the modified PDF. Selecting Save will need to be repeated if changes are made following a Save and prior to selecting Add.

  15. If necessary, you can view the modified version of the selected PDF with the bundled viewer either by selecting Preview or by pressing Alt + P on the keyboard.

  16. If no further changes need to be made and you wish to upload the file, select Add or press Alt + A on the keyboard. To cancel and discard the modified PDF, select Cancel.
