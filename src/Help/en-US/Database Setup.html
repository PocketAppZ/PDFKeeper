Database Setup

PDFKeeper requires a relational database management system for document storage. It can be implemented for use in single user or multi user environments. If implementing for use in a multi user environment, it is recommended that the database be installed on a dedicated system that is capable of handling the expected load.

At this time, PDFKeeper has only been designed for use with Oracle Database. Future releases may include support for additional relational database management systems based on user request and the feasibility of implementation. For a database to be considered, please open an new issue on the Issue Tracker.

As an alternative to the fully supported, paid editions of Oracle Database, Oracle offers an easy to install, Oracle Database Express Edition (XE) which is perfect for single user or small work group environments that do not need a fully supported edition or for those who just wish to try out PDFKeeper – all completely free!
